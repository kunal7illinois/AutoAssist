STEP 17. Check the input signal (by using the Pulse
Check).
Check the input signal of the liftgate latch switch <vehicles with
theft-alarm system> or liftgate light switch <vehicles without
theft-alarm system>.
CAUTION
To prevent damage to scan tool MB991502, always turn the
ignition switch to the "LOCK" (OFF) position before con-
necting or disconnecting scan tool MB991502.
(1) Connect scan tool MB991502 to the data link connector.
(2) Operate the MUT-II as follows:
1. Select "SYSTEM SELECT."
2. Select "SWS."
3. Select "PULSE CHECK."
(3) When the liftgate is opened and closed, check if scan tool
MB991502 sounds or not.
