23A-196 AUTOMATIC TRANSAXLE
AUTOMATIC TRANSAXLE DIAGNOSIS
STEP 1. Using scan tool MB991502, check the A/T
diagnostic trouble code.
CAUTION
To prevent damage to scan tool MB991502, always turn the
ignition switch to the "LOCK" (OFF) position before con-
necting or disconnecting scan tool MB991502.
(1) Connect scan tool MB991502 to the data link connector.
(2) Turn the ignition switch to the "ON" position.
(3) Read the A/T diagnostic trouble code.
(4) Turn the ignition switch to the "LOCK" (OFF) position.
Q:Is the A/T diagnostic trouble code number "54" output?
YES : Refer to P.23A-233, code number 54: A/T control
relay system.
