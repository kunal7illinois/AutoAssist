AUTOMATIC TRANSAXLE DIAGNOSIS
STEP 1. Using scan tool MB991502, check data list item 21:
Crankshaft Position Sensor.
CAUTION
To prevent damage to scan tool MB991502, always turn the
ignition switch to the "LOCK" (OFF) position before con-
necting or disconnecting scan tool MB991502.
(1) Connect scan tool MB991502 to the data link connector.
(2) With the gear selector lever in the "P" position, start the
engine and run at idle.
(3) Set scan tool MB991502 to data reading mode for item 21:
Crankshaft Position Sensor.
• When the accelerator pedal is not depressed (throttle
valve is fully closed) the display should be "600 to 900 r/
min."
