RADIO PRESET MEMORY IS LOST
NOTE: If only the XM stations are lost, go to XM RADIO PRESET
MEMORY IS LOST .
1. Set each of the radio station preset buttons.
Do each of the buttons set properly?
YES -Go to step 2.
NO -
With navigation: Replace the navigation unit (see NAVIGATION UNIT
(cid:1)
REMOVAL/INSTALLATION ).
Without navigation: Replace the audio unit (see AUDIO UNIT
(cid:1)
REMOVAL/INSTALLATION).
2. Turn the ignition switch to LOCK (0) for 1 minute, then turn it back to ON (II).
3. Test the preset buttons for proper recall operation.
Do the preset buttons recall the set radio stations?
YES -System is normal at this time. Check the connections at the navigation
unit or audio unit.
NO -
With navigation: Replace the navigation unit (see NAVIGATION UNIT
(cid:1)
REMOVAL/INSTALLATION ).
