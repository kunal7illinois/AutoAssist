8.4 On-screen fault messages
In the event of a system fault, the fault is stored in the fault memory
and/or a fault message appears on the display of the navigation
system with CD drive control unit -J401- .
At present the following fault messages can be displayed.
Indicated on display:
CD/DVD is unreadable
Please check the CD/DVD
This fault message occurs in the following situations:
♦ The navigation DVD cannot be read.
♦ DVD is defective.
♦ The DVD drive is defective.
– Check whether the media drive can read a replacement DVD.
If the same fault message occurs:
– Perform adaption of channel 41.
