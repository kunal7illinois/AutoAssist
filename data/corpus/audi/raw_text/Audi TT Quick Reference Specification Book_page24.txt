General, Technical Data
Engine number – Location – 2.0L – CCTA
The engine number (engine code and serial number) is located at
the left on the engine/transmission partition. The engine code is also
stamped on the right side of the cylinder head and on the cylinder
block. In addition, a sticker with the engine code and serial number
is affixed to the timing belt cover. The engine code is also on the
vehicle data plate.
