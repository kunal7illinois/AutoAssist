AX–157
U660E AUTOMATIC TRANSAXLE – AUTOMATIC TRANSAXLE SYSTEM
DTC U0100 Lost Communication with ECM/PCM "A"
DESCRIPTION
The engine control unit intercommunicates with the TCM with the Controller Area Network (CAN).
If there is a problem in this intercommunication, the TCM sets a DTC.
