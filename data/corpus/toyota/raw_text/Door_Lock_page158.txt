DOOR LOCK – SMART KEY SYSTEM
• Driver door unlock mode:
When the driver door's touch sensor is touched,
only the driver door unlocks. When another touch
sensor is touched, all doors unlock.
• All door unlock mode:
When the touch sensor is touched, all doors
unlock.

wireless door lock buzzer and combination meter to
inform the user that the mode has been switched.
