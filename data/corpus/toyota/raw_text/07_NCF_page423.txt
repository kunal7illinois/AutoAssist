BODY ELECTRICAL - SMART KEY SYSTEM BE-89
SMART KEY SYSTEM
(cid:2)DESCRIPTION
(cid:1) This system is optional equipment on the XLE grade (except Mexican package model).
(cid:1) The smart key system not only has a wireless door lock remote control function and engine immobilizer
function, but by carrying the key the following functions (entry function and push button start function)
are also possible without having to use a key or transmitter button. It is an extremely convenient system.
- The engine can be started by simply pressing the engine switch while depressing the brake pedal (Push
Button Start Function)
- Door unlock/lock (Entry Unlock/Entry Lock Functions).
- The trunk can be opened (Trunk Open Function).
- Wireless door lock control function.
