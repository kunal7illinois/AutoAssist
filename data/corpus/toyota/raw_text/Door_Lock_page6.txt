DOOR LOCK – POWER DOOR LOCK CONTROL SYSTEM
Function Outline
When the conditions listed below are met consecutively, this function
causes all the doors to be automatically locked.
• The ignition switch is on (IG).
Shift-linked automatic door lock function (A/T)
• All doors are closed.
• The shift lever is moved out of the P position.
• Any of the doors are unlocked.
When the ignition switch is on (IG), if moving the shift lever to the P
Shift-linked automatic door unlock function (A/T) position from any other position, all the doors will be automatically
unlocked.
When the door is locked, this function enables the door to be
One-motion open unlocked by pulling the inside handle of the driver's or front
passenger's door.
This function is provided to unlock the driver's door by turning the key
2-step unlock function cylinder once and to unlock the remaining doors by turning it the
second time.
*1: w/ Smart Key System
*2: w/o Smart Key System
