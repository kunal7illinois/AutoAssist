2AZ-FE STARTING – STARTER
REASSEMBLY
HINT:
Use high-temperature grease to lubricate the bearings,
gears, return spring and steel ball when assembling the
starter.
1. INSTALL PLANETARY GEAR
Apply Grease (a) Apply grease to the planetary gears and pin parts of
the planetary shaft.
(b) Install the 3 planetary gears.
A081656E01
2. INSTALL STARTER ARMATURE ASSEMBLY
(a) Apply grease to the plate washer and the armature
shaft.
Snap Ring Pliers
(b) Install the starter armature to the starter commutator
end frame.
(c) Using snap ring pliers, install the plate washer and a
new snap ring.
A082440E01
(d) Using a vernier caliper, measure length of the snap
ring.
Length
Maximum length:
5.0 mm (0.197 in.)
If the length is greater than the maximum, replace it
with a new snap ring.
A058810E01
3. INSTALL STARTER COMMUTATOR END FRAME
COVER
(a) Install the end frame cover to the commutator end
frame.
