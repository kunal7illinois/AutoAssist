SUPPLEMENTAL RESTRAINT SYSTEM – OCCUPANT CLASSIFICATION SYSTEM
DATA LIST / ACTIVE TEST
HINT:
By accessing the DATA LIST displayed on the intelligent
tester, you can perform such functions as reading the values
of switches and sensors without removing any parts. Reading
the DATA LIST is the first step in troubleshooting is one
method to save labor time.
1. DATA LIST FOR OCCUPANT CLASSIFICATION ECU
(a) Connect the intelligent tester to the DLC3.
(b) Turn the ignition switch on (IG).
(c) Turn the tester ON.
(d) Enter the following menus: DIAGNOSIS / OBD/
MOBD / OCCUPANT DETECT / DATA LIST.
(e) Check the values by referring to the table below.
