EI–79
ENGINE IMMOBILISER – ENGINE IMMOBILISER SYSTEM (w/o Smart Key System)
DTC B2784 Antenna Coil Open / Short
DESCRIPTION
The transponder key coil is built into the transponder key amplifier and receives a key code signal from
the transponder chip in the key. This signal is amplified by the amplifier, then it is output to the transponder
key ECU assembly.
