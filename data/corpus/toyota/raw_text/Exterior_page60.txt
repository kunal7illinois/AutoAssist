ET–57
EXTERIOR – REAR DOOR WINDOW FRAME MOULDING
INSTALLATION
1. INSTALL REAR DOOR UPPER WINDOW FRAME
MOULDING
(a) Install the rear door upper window frame moulding
to the door frame.
(b) Using an air riveter or hand riveter with a nose
piece, install the rear door upper window frame
moulding with 7 new rivets.
HINT:
If the rivet cannot be cut, pull it once and cut it.
