LIGHTING – AUTOMATIC LIGHT CONTROL SENSOR
REMOVAL
1. REMOVE NO. 1 DEFROSTER NOZZLE GARNISH
(See page IP-32)
2. REMOVE AUTOMATIC LIGHT CONTROL SENSOR
(a) Disengage the 2 claws and remove the automatic
light control sensor.

INSTALLATION
1. INSTALL AUTOMATIC LIGHT CONTROL SENSOR
(a) Engage the 2 claws and install the automatic light
control sensor.
2. INSTALL NO. 1 DEFROSTER NOZZLE GARNISH (See
page IP-48)
