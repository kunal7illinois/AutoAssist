DESCRIPTION
This DTC is output when the ID code box does not respond to a verification request signal from the ECM.
HINT:
• If the engine switch is turned off immediately after an engine start, this DTC is also detected.
• If this DTC is detected simultaneously with DTC B2799, troubleshoot DTC B2799 first.
