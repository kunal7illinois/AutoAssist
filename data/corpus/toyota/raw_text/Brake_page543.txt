BR–66
BRAKE – REAR BRAKE
7. INSTALL REAR DISC BRAKE PISTON
(a) Apply lithium soap base glycol grease to the rear
disc brake piston and a new cylinder boot.
(b) Install the cylinder boot to the rear disc brake piston.
(c) Install the piston to the rear disc brake cylinder sub-
assembly.
NOTICE:
Do not install the piston forcibly in the disc
brake cylinder.
:Lithium soap base glycol grease

8. INSTALL CYLINDER BOOT
Set Ring Piston (a) Install the cylinder boot to the rear disc brake
cylinder sub-assembly.
BR NOTICE:
• Install the boot securely onto the grooves of
the cylinder and the piston.
• Do not damage the cylinder boot.
(b) Using a screwdriver, install the set ring.
Cylincer Cylinder Boot CAUTION:
Be careful not to damage the boot.

9. INSTALL REAR DISC BRAKE BUSHING DUST BOOT
(a) Apply lithium soap base glycol grease to the seal
surface of 2 new rear disc brake bushing dust boots.
(b) Install the 2 rear disc brake bushing dust boots to
the rear disc brake cylinder mounting.
