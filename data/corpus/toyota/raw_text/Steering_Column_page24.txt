SR–22
STEERING COLUMN – ELECTRIC STEERING LOCK
DTC B2782 Power Source Control ECU Malfunction
DESCRIPTION
The main body ECU (*) controls the power supply to activate the steering lock motor. This prevents the
steering from being locked while the vehicle is moving.
HINT:
*: The power source control ECU is not a physical part. This code refers to the power source control
function performed by the main body ECU.
