EI–89
ENGINE IMMOBILISER – ENGINE IMMOBILISER SYSTEM (w/o Smart Key System)
DTC B2799 Engine Immobiliser System
DESCRIPTION
This DTC is output when the ECM detects errors in communication with the transponder key ECU, and in
the communication lines.
This DTC is also output when an engine start is attempted while the ECU communication ID between the
transponder key ECU and the ECM is different.
Before troubleshooting for this DTC, make sure that there is no DTC detected in the transponder key
ECU. If there is any key code related DTC detected in the transponder key ECU, repair it first.
