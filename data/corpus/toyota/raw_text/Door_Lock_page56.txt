DL–56
DOOR LOCK – POWER DOOR LOCK CONTROL SYSTEM
Door Key Lock / Unlock Switch Circuit
DESCRIPTION
The door lock and unlock switch is built into the door lock assembly.
This switch is ON when the door key is in the unlock position and OFF when the door key is in the lock
position. It is used as one of the operating conditions for the key confinement prevention function.
