CC–36
CRUISE CONTROL – CRUISE CONTROL SYSTEM
Cruise Main Indicator Light Circuit
DESCRIPTION
• The ECM detects a cruise control switch signal and sends it to the combination meter through CAN.
Then the CRUISE main indicator light comes on.
• The CRUISE main indicator light circuit uses CAN for communication. If there is a malfunction in this
circuit, check for DTCs in the CAN communication system before troubleshooting this circuit.
